<p align="center">
  <b>CYBERIX</b> is an advanced, automated weather data extraction tool built with Python. It seamlessly interacts with the Open-Meteo REST API to fetch live environmental data and maintain structured local logs.
</p>

<img src="https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/rainbow.png" width="100%" />

## ✨ Features
- 🔄 **Real-Time Data Extraction**: Fetches live temperature and wind speed via robust API calls.
- 📝 **Automated Logging**: Generates timestamped reports and appends them to a continuous log file.
- ⚡ **Lightweight & Fast**: Optimized for Linux environments like Termux.
- 🛡️ **Error Handling**: Built-in HTTP status code validation.

<img src="https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/rainbow.png" width="100%" />

## 🚀 Installation & Usage

### 1. Clone the Repository
`git clone https://github.com/basak-works/Termux.git`
`cd Termux/CYBERIX`

### 2. Install Dependencies
`pip install requests`

### 3. Run the Bot
`python api_bot.py`
